Coloring Text Elements

To color text, use the color name as the class.

H1 class="moonlight"

H2 class="guardsman"

H3 class="turquoise"

<h1 class="moonlight">Your Text</h1>
				
<h2 class="guardsman">Your Text</h2>
				
<h3 class="turquoise">Your Text</h3>

Coloring Block Elements

To color background sections and div blocks, use bg-'color name' as the class.

<section class="content-block bg-finn">Your content...</section>
